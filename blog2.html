GSOC-A Beginners Guide

What is Gsoc?

"Google Summer of Code is program to promote open-source development. It is organized by Google every summer since last decade. Students contribute to open-source repositories and get paid by Google in return. It's a great way to spend summer vacation.

Eligibility Criteria

  • You must be at least 18 years of age
  • You must currently be a full or part-time student (or have been accepted for the fall term) at an accredited university as of the student acceptance date
  • You must be eligible to work in the country you will reside in during the program
  • You have not already been accepted as a Student in GSoC more than once
  • You must reside in a country that is not currently embargoed by the United States. See Program Rules for more information.

Stipend

Earlier it is used to be $3000 but for 2021 the stipend is reduced by half so you will get $1500.

Things to remember

  • Start Early- So that you are able to submit draft proposals, before you submit the final one. Generally the projects, require some industrial tools for project management hence getting in touch with project mentors early and discussing these details in the beginning and putting the details about the same in the proposal will help you out.
  • Learn to minimize- You should know your priorities, don’t just run after all the projects,don’t aim for projects which you wont be able to handle.
  • Your proposal must contain -
  • A well written summary of your project
  • A detailed description of the problem, you want to solve
  • Description of any feature which you would like to add
  • Include any work experience or project you have worked on in the past
  • Write your vision for the project
