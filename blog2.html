Array

WHAT IS AN ARRAY?

An array is one of the most fundamental data structures.It stores data in a sequential manner.

A pair of square brackets [] represents an array in JavaScript. All the elements in the array are comma(,) separated.

An array can hold values of different types. For example, you can have an array that stores the number and string, and boolean values.

var person = ["John", "Wick", 32, 87];

Accessing elements of an array

Each value is called an element specified by an index.

We can access an element of the array by referring its index.The index in array starts from 0.[0] is the first element. [1] is the second element.

Creating JavaScript arrays

an array can be created by two ways:

  1. 1. Using an array literal

    2. The easiest way to create an array is by using an array literal []. For example,

    const array1 = ["eat", "sleep","conquer","repeat"];

  2. 2. Using the new keyword

    3. The another way to create an array is by using new kwyword. For example,

    const array2 = new Array("eat", "sleep","conquer","repeat");

Getting The Length Of An Array

The length property returns the length of an array, which is the total number of elements contained in the array. Array length is always greater than the index of any of its element.

For example: var myArray = [ 'hello', 'world', '!'];
console.log(myArray.length);
It will show output as 3 in console which is the actual length of this array
