A Basic Introduction To Markdown

What Is Markdown?

Markdown is a plain text to html conversion tool aimed at making writing for the internet easier.The main philosophy behind markdown is that it allows to write in easier to read easier to write plain text format and then convert it structutrally into valid html.

Why to use Markdown?

It converts to HTML easily

Now, if you’re going to be writing something into HTML, you should write HTML. But if you’re, say, writing an email or a readme file where you need HTML’s formatting options but not all of its features, Markdown is perfect in order to write same in HTML you have to learn its syntax,the hassle of opening and closing tag plus html in itself is a very vast which comes with its own learning curve so if you are a beginner and you haven't used HTMl before it can be overwhelming when you just want to write an email or a readme file or you just want to bold a text or highlight something or maybe just adding bullets in list.

Markdown is everywhere. Websites like Reddit and GitHub support Markdown, and lots of desktop and web-based applications too supports Markdown.

How to write Markdown syntax?

let’s learn a few basic formatting techniques that will help you to get started with Markdown.

1.Headings

Any line which is prefixed with a # symbol is converted into a heading. The number of hashes indicates the level of the heading. For example: one hash is converted to an h1 heading, two hashes to an h2 and so on (till h6).
# This is an h1 heading
## This is an h2 heading

2. Emphasis

If you want to put emphasis on a word, make use of asterisk symbol. Single asterisk will italicize the text and a double asterisk will make the word bold.
*This text will be italic*
**This text will be bold**

3. Lists

For a bullet (unordered) list in Markdown, just prefix each word with an asterisk.
*this is a list
*bread
*butter
For a numbered (ordered) list, just put numbers instead of an asterisk.
